History

CSS was first proposed by Håkon Wium Lie on October 10, 1994. At the time, Lie was working with Tim Berners-Lee at CERN. Several other style sheet languages for the web were proposed around the same time, and discussions on public mailing lists and inside World Wide Web Consortium resulted in the first W3C CSS Recommendation (CSS1) being released in 1996. In particular, a proposal by Bert Bos was influential; he became co-author of CSS1, and is regarded as co-creator of CSS. (Wikipedia, 2020)

CSS Essentials: CSS syntax

  • CSS rule 
    
					h1 {
						color: green;
					}
  • h1 - selector, which element to selector to set styling
  • color: property
  • orange: value of property
  • { color: orange } - declaration. We can have as much declarations as we want

CSS Selectors

  • Type (h1, a, div): name of elements, and it attached to all type of element on the page. It is the most basic css selector
  • Descendant (div a {}): more appropriate selector. We can specify elements, or its group by specified how they are nested. Opisanie potomstwa i rodzica elementów
  • Class: specified attributes. Reuse same style in many places.

cd. CSS Selectors

  • Mix descendant and class selectors - w zależności gdzie dany element się znajduje możemy go stylować w różny sposób; 
    
						p {color: orange}
						footer {color: green}
						footer p {color: blue}

cd. CSS Selectors: remember

descendent selectors are more specific, deeped elements are more specified; class are more specific than descendant; class with descendant are more specific than class. if element have two rules with same specifically then the order of the rule are count.

The Cascade

  • Cascade: like waterfall. Default style by browser. Adding new stream in cascade rewrite default browser’s style.
  • Inheritance: allow to set style from parent elements to children element. Some value of property can’t be inherit, like color (more about it later).
  • Specificity: specyficzność. Obliczanie specyficzności, aby określić, która reguła dla jednego elementu wygrywa i będzie go stylować.

Box Model

Every elements on the web page are a rectangle or a box. Styling mean to decorate, and manipulate boxes in any way that we can imagine 


							box-a {
								background-color: orange;
								padding: 30px;

  • Box: area which take space on the webpage
  • Padding: create space to inside of that box
  • Border: add outline on the outer edge of the box
  • Margin: create space around outside of the box

How to control total width 


					box-a {
						width: 20px;
						height: 20px;
					 }

Compare widths 


				box-a {
					width: 20px;
					height: 20px;
				 }
				box-b {
					width: 20px;
					height: 20px;
					padding: 20px;
					border: 10px;
				 }

box-a = 20px;

box-b = 20px + 20px left padding + 20px right padding + 10px left border + 10px right border = 60px
