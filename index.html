History

CSS was first proposed by Håkon Wium Lie on October 10, 1994. At the time, Lie was working with Tim Berners-Lee at CERN. Several other style sheet languages for the web were proposed around the same time, and discussions on public mailing lists and inside World Wide Web Consortium resulted in the first W3C CSS Recommendation (CSS1) being released in 1996. In particular, a proposal by Bert Bos was influential; he became co-author of CSS1, and is regarded as co-creator of CSS. (Wikipedia, 2020)

Cascading Style Sheet (CSS)

HTML and CSS purpose

  • HTML - define what is a structure and content
  • CSS - define how something should look like inspiration

How to add CSS style to a page?

  • Add link element to a header. In Header html keeps all information about content of the page. Content of a page is in body element.
  • rel: relationship with stylesheet (type of file).
  • href: path to the file with css code.

CSS syntax

  • CSS rule 
    
					h1 {
						color: green;
					}
  • h1 - selector, which element to selector to set styling
  • color: property
  • orange: value of property
  • { color: orange } - declaration. We can have as much declarations as we want

CSS Selectors

  • Type (h1, a, div): name of elements, and it attached to all type of element on the page. It is the most basic css selector
  • Descendant (div a {}): more appropriate selector. We can specify elements, or its group by specified how they are nested - parent and children relations.
  • Class: specified attributes. Reuse same style in many places.

cd. CSS Selectors

  • Mix descendant and class selectors - depend from where some element exist in html structure we can styled it in deferent ways. 
    
						p {color: orange}
						footer {color: green}
						footer p {color: blue}

Remember

descendent selectors are more specific, descendent elements are more specified; class are more specific than descendant; class with descendant are more specific than class. if element have two rules with same specifically then the order of the rule are count.

The Cascade

  • Cascade: like waterfall. Default style by browser. Adding new stream in cascade rewrite default browser’s style.
  • Inheritance: allow to set style from parent elements to children element. Some value of property can’t be inherit, like color (more about it later).
  • Specificity [specyficzność]: We can counting specificity for find out which rules are attached to some element and influence how it look like.

Every elements on the web page are a rectangle or a box. Styling mean to decorate, and manipulate boxes in any way that we can imagine

  • Box: area which take space on the webpage
  • Padding: create space to inside of that box
  • Border: add outline on the outer edge of the box

How to control total width and height? 


					box-a {
						width: 20px;
						height: 20px;
					 }

Compare total widths 


				box-a {
					width: 20px;
					height: 20px;
				 }
				box-b {
					width: 20px;
					height: 20px;
					padding: 20px;
					border: 10px;
				 }

box-a = 20px;

box-b = 20px + 20px left padding + 20px right padding + 10px left border + 10px right border = 60px

Compare total heights 


				  box-a {
					  width: 20px;
					  height: 20px;
				   }
				  box-b {
					  width: 20px;
					  height: 20px;
					  padding: 20px;
					  border: 10px;
				   }

box-a = 20px;

box-b = 20px + 20px top padding + 20px bottom padding + 10px top border + 10px bottom border = 60px

Something is wrong?

Why is box-b not the same size as box-b?

Solution

Use box-sizing properties in universal selector to control width and height of box

Default


					* {
						box-sizing: content-box 
					}

Modern


						* {
							box-sizing: border-box 
						}

Box-sizing doesn’t count a margin properties!

Overlapping effects with positioning properties

Positioning your elements (boxes), so they can lay on each other. We will talk about position property with relative and absolute value.

Positioning

  • position absolute: position element without conjunction with the rest elements, only with the parents with position relative. If there is no one, the absolute position is conjunction with view screen.
  • battle of overlapping: z-index. If few elements have a position relatives, and they will be overlapping on each other, then we can decide which one should be on top by using z-index property with any negative or positive values: z-index: 1; or z-index: -1;

Color units

  • Hexadecimal RGB
  • RGB
  • RGBA
  • May be is more?

Hexadecimal

Hexadecimal RGB value: Six value from 0 - 9 and a - f. Each pair represent a variation of colors: red; green; blue with 256 combination (16 * 16 = 256);

Example HEX

#ff9999

RGB & RGBA

RGB & RGBA: red-green-blue /-alpha: value for each color in notation of 255. “A” value is for transparency from 0 to 1.

Example RGB

rgb(255, 153, 153)

Example RGBA

rgba(255, 153, 153, .5)

Background transparency

  • What is it?
    div { background-color: rgba(255, 99, 71, 0.5); }
  • How to set background transparency?

    Use rgba and set alpha value in background-color property.
